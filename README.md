# TraceMoe vk bot
### Simple bot using [aiomoe](https://github.com/FeeeeK/aiomoe) library and [vkbottle](https://github.com/timoniq/vkbottle) framework
## Installation
```bash
git clone https://github.com/FeeeeK/TraceMoeVKBot tm_bot
cd tm_bot
pip install poetry
poetry install
```
Then just rename `.env.example` to `.env`, add the token there and write:
```bash
python .
```
