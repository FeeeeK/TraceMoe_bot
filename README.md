# TraceMoe vk bot
### Simple bot using [tracemoe](https://github.com/Ethosa/tracemoe) library and [vkbottle](https://github.com/timoniq/vkbottle) framework
## Installation
```sh
git clone github.com/feeeek/TraceMoe_bot
cd TraceMoe_bot
pip install -r requirements.txt
```
Next, just add the VK group token to `config.json` and write:
```sh
python main.py
```
